# portman-netex

Express-based server to use communicate with other components via [ZeroMQ](http://zeromq.org/),
then cache the results in [Redis](https://redis.io/).

This allows micro-services to be written in any language that supports ZeroMQ, while in the same
time, that best fits the service it provides.
